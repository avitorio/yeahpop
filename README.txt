Display Your Recent WooCommerce Sales To Increase Your Conversion Rate.

== Description ==

YeahPop shows the last orders for a product as small, non-intrusive popup notifications.
The plugin only work on product pages.
The notification shows the buyer's name, location and how long ago they purchased.

== Installation ==

1. Upload `yeahpop.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in wordpress
3. Done!

== Frequently Asked Questions ==

= How many orders will Yeahpop show =

YeahPop takes the last 5 orders for a product and loops them as notifications.
Each notification lasts 8 seconds and appears within 8 seconds intervals.
